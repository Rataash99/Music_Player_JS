## Overview

### The challenge

Users should be able to:

- Play selected song
- Toggle Theme (Dark / Light)
- filter songs according to genre
- create PlayList
- Add songs to PlayList
- Delete songs from PlayList
- Search PlayList
- Play prev and next song present in selected genre
